Face Recognition Attendance System

This project was an exploration into computer vision and machine learning. I built a proof-of-concept system that uses a webcam to detect faces and match them against a database of known individuals to log attendance automatically. The core of the system was built in Python, leveraging the OpenCV library for facial recognition algorithms. A simple GUI was created with Tkinter to manage the user database.
